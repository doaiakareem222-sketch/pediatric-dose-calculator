# PediaDose AI

Smart Pediatric Dose Calculator

## About
PediaDose AI is a web application designed to help healthcare professionals calculate pediatric medication doses accurately based on weight, age, and trusted medical references.

## Features
- Pediatric dose calculator
- Weight-based dosing
- Syrup mL calculator
- Dose safety alerts
- Drug database
- Arabic & English support
